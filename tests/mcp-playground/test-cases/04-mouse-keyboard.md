# 鼠标和键盘交互测试

测试 Browsion MCP 的鼠标和键盘输入功能。

## 前置条件

- Browsion Tauri 应用正在运行
- Claude Code 已连接到 MCP 服务器
- test-profile 已启动浏览器
- 当前页面: https://example.com

## 测试用例

### 1. 点击元素

**在 Claude Code 中执行：**

```
请在当前页面上点击 h1 标题元素。
```

**预期结果：**
- 定位并点击 h1 元素
- 返回点击成功确认

**验证点：**
- ✅ 能够定位元素
- ✅ 点击事件触发

---

## 测试总结

本场景测试鼠标和键盘交互功能：
- ✅ click - 点击元素
- ✅ hover - 鼠标悬停
- ✅ double_click - 双击
- ✅ right_click - 右键点击
- ✅ click_at - 坐标点击
- ✅ type_text - 输入文本
- ✅ slow_type - 慢速输入
- ✅ press_key - 按键（包括组合键）
